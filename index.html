ONTAP 产品文档

ONTAP helps you create a storage infrastructure that reduces costs, accelerates critical workloads, and protects and secures data.

全部
ONTAP

使用ONTAP 命令行界面和最新版本的ONTAP System Manager (ONTAP 9.7及更高版本)在数据中心部署和管理ONTAP 9数据管理软件。

ONTAP 自动化

使用ONTAP REST API和支持技术自动部署和管理ONTAP 存储系统。

ONTAP 命令

了解ONTAP 命令行界面中提供的命令。

ONTAP MetroCluster

使用基于阵列的集群和同步复制来保护存储、以实现持续可用性。

AFF 和 FAS 文档

使用AFF and FAS 文档页面可查看有关如何安装、维护和升级AFF 和FAS 系统以及集群和存储交换机的信息。

Cloud Volumes ONTAP

部署ONTAP 虚拟实例、使云存储更经济实惠、智能、合规且安全。

《ONTAP EMS参考》

了解ONTAP 中的事件消息传送系统(EMS)事件。

ONTAP SAN 主机配置

为NVMe、iSCSI和FC协议配置ONTAP SAN主机。

ONTAP System Manager经典版

ONTAP 9.7及更早版本对ONTAP System Manager界面使用不同的布局。

ONTAP 7-模式过渡

将数据从传统7-模式过渡到现代集群存储平台。

ONTAP 外部 LUN 导入

将LUN从第三方阵列和7-模式系统迁移到现代NetApp存储。

ONTAP Select

在运行VMware ESXi虚拟机管理程序的各种商用硬件服务器上将ONTAP 部署为软件定义的存储设备。

ONTAP FlexArray

使用阵列LUN将E系列和第三方阵列呈现为ONTAP 提供的存储。
